# TravelTrucks - Camper Rental Service

Веб-додаток для оренди кемперів компанії "TravelTrucks". Проєкт дозволяє переглядати каталог транспортних засобів, фільтрувати їх за характеристиками, додавати до обраного та бронювати.

## Основні функції

- **Каталог кемперів**: Перегляд доступних варіантів з пагінацією.
- **Фільтрація**: Пошук за локацією, типом кузова та обладнанням (AC, кухня, TV тощо).
- **Детальна сторінка**: Опис характеристик, галерея та відгуки користувачів.
- **Обране**: Можливість зберігати кемпери (дані зберігаються після оновлення сторінки).
- **Бронювання**: Форма для замовлення з нотифікацією про успіх.

## Технологічний стек

- **Framework**: Next.js (App Router)
- **Language**: TypeScript
- **State Management**: Zustand
- **API Client**: Axios
- **Styling**: CSS Modules
